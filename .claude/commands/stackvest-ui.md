Review and apply the StackVest design language for the task at hand.

Read the full skill at `skills/stackvest-ui/SKILL.md` and follow all instructions there, including:

1. Checking `src/index.css` for available CSS variables before writing any color or spacing values.
2. Applying the Sans/Mono typography split (Inter for UI, JetBrains Mono for financial figures).
3. Using `8px` border radius on interactive elements, `4px`/`6px` for small accents.
4. Verifying both Light and Dark mode appearances.
5. Running the checklist at `skills/stackvest-ui/checklist.md` before finishing.
